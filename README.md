# Final project for Web application - Group 11 - FunPop

---
## Project Overview
Brief description of the project:
> FunPop là một mạng xã hội dành riêng cho những người yêu âm nhạc, đam mê văn hoá fandom và thích sưu tầm các món goods độc quyền: album, photocard, lightstick, vé concert, merchandise giới hạn…

FunPop được tạo ra để:

- Kết nối những người có chung gu âm nhạc và thần tượng

- Tạo một không gian an toàn để chia sẻ bộ sưu tập, khoe goods, kể câu chuyện đằng sau từng món đồ
