# FUNPOP – Social Network Web Application
---
## Project Overview
> FUNPOP is a web-based social networking application that allows users to connect, share posts, and interact with others in a fun and friendly environment. This project is developed as part of the Web Application Development course.

## Tech Stack

### Frontend
- **HTML5** - Semantic markup
- **Bootstrap 5.3.3** - Responsive UI framework
- **Bootstrap Icons 1.11.3** - Icon library
- **CSS3** - Responsive styling (Mobile-first design)
- **JavaScript (ES6+)** - Interactive features
  - AJAX for asynchronous requests
  - Infinite scroll pagination
  - Real-time notifications
  - Mention autocomplete functionality
  - Comment system

### Backend
- **PHP 8.4** - Server-side logic with latest features
- **Router** - Custom routing system for API endpoints
- **MVC Architecture** - Organized code structure with Controllers, Models, and Views

### Database
- **MySQL** - Relational database
- **Database Migrations** - Version control for schema changes

### Third-Party Services & Libraries
- **AWS S3** - Cloud storage for media files
- **Firebase** - Real-time messaging and notifications
- **PHPMailer** - Email functionality
- **Composer** - PHP dependency management

### Core Features
- **Avatar System** - User profile image handling
- **Media Upload** - Post media management (images, videos)
- **Hashtag Support** - Content categorization and discovery
- **User Badge System** - User recognition and gamification
- **Notification System** - Real-time and in-app notifications
- **Search Functionality** - Post and user search capabilities
- **Saved Posts** - Bookmark system for users

### Real-Time Chat Features
- **Firebase Integration** - Real-time messaging with instant message delivery
- **Direct Messaging** - 1-on-1 private conversations
- **Message Attachments** - Share files in conversations
- **Chat Bootstrap API** - Session initialization for real-time message sync
- **Chat Notifications** - Real-time notification badges for new messages
- **Message History** - Persistent storage of conversations
- **User Online Status** - Display user availability in chat
